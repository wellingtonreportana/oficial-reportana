---
title: "Dados de Pedidos"
description: "*Variáveis disponíveis para imprimir dados de pedidos em mensagens.*"
---

Estão disponíveis para uso somente em mensagens destinadas a pedidos. Para utilizar dados de pedidos em suas mensagens, você _deve categorizar a mensagem_ com o **Tipo PEDIDO**. Ao fazer isso, nosso sistema realizará a leitura dos dados de pedidos e permitirá seu uso adequado.

**Dados Gerais do Pedido**

| **Nome**                                     | **Variável**                                    | **Saída**                                                                 | **Tipo** |
| -------------------------------------------- | ----------------------------------------------- | ------------------------------------------------------------------------- | -------- |

| Linha digitável do boleto / Pix Copia e Cola | `{{ order.billet_line }}`                       | Código de Pagamento                                                       | String   |
| Link do boleto/pagamento                     | `{{ order.billet_url }}`                        | link.pagamento/exemPLO                                                    | String   |
| Moeda                                        | `{{ order.currency }}`                          | BRL                                                                       | String   |
| Número do pedido                             | `{{ order.number }}`                            | 11205                                                                     | String   |
| Valor subtotal                               | `{{ order.subtotal_price }}`                    | 9.90                                                                      | Number   |
