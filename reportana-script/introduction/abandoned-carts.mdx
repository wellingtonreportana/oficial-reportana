title: "Dados de Carrinhos Abandonados"
description: "*Variáveis disponíveis para imprimir dados de carrinhos abandonados em mensagens.*"
---

Essas variáveis estão disponíveis para uso somente em mensagens destinadas a carrinhos abandonados. Para utilizar dados de carrinhos em suas mensagens, você _deve categorizar a mensagem_ com o **Tipo CARRINHO ABANDONADO**. Ao fazer isso, nosso sistema realizará a leitura dos dados de carrinhos abandonados e permitirá seu uso adequado.
