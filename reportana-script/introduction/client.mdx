---
title: "Dados do Cliente"
description: "*Variáveis disponíveis para imprimir dados de clientes em mensagens.*"
---

Indicamos utilizar sempre que precisar exibir informações sobre o cliente. Essas informações são as mesmas atreladas ao pedido/carrinho do cliente. Portanto, é possível utilizar essas variáveis tanto em mensagens destinadas a pedidos quanto em mensagens destinadas a carrinhos abandonados.

| **Nome**           | **Variável**       | **Saída**       | **Tipo** |
| ------------------ | ------------------ | --------------- | -------- |
| Email              | `{{ email }}`      | teste@email.com | String   |
| Número de telefone | `{{ phone }}`      | 55999999999     | String   |
| Primeiro nome      | `{{ first_name }}` | João            | String   |
| Último nome        | `{{ last_name }}`  | Teste           | String   |
| Nome completo      | `{{ name }}`       | João Teste      | String   |
