### Introdução

O filtro `lower` é utilizado para converter todos os caracteres de uma string para minúsculas. Isso é útil quando você precisa padronizar a formatação de texto, por exemplo, ao exibir informações de forma consistente ou ao realizar comparações de texto.

### Exemplo de uso

Exemplo de como usar o filtro `lower` para converter uma string para minúsculas:

```twig
{{ 'Hello, World!'|lower }}
```
