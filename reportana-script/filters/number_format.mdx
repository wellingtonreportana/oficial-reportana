### Introdução

O filtro `number_format` é usado para formatar números de acordo com um número específico de casas decimais, separadores de milhar e separadores decimais. Ele é útil para exibir valores numéricos de forma padronizada, como preços ou quantidades, em uma interface amigável.

### Exemplo de uso

Exemplo de como utilizar o filtro `number_format` para formatar um número com duas casas decimais, utilizando a vírgula como separador de milhar e o ponto como separador decimal:

```twig
{{ 1234567.89|number_format(2, '.', ',') }}
```
