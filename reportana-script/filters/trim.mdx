### Introdução

O filtro `trim` é usado para remover espaços em branco (ou outros caracteres) do início e do final de uma string. Ele é útil para limpar dados de entrada ou garantir que strings sejam processadas sem espaços indesejados.

### Exemplo de uso

Exemplo de como utilizar o filtro trim para remover espaços em branco de uma string:

```twig
{% set text = '   Reportana   ' %}
{{ text|trim }}
```
