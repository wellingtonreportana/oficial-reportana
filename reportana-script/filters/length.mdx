### Introdução

O filtro `length` é utilizado para obter o número de elementos em uma lista ou a quantidade de caracteres em uma string. Ele é útil para determinar a extensão de arrays, objetos ou strings, o que pode ser essencial para controle de fluxo e formatação de dados.

### Exemplo de uso

Exemplo de como usar o filtro `length` para obter o número de elementos em uma lista e a quantidade de caracteres em uma string:

```twig
{# Obtendo o número de elementos em uma lista #}
{{ ['Apple', 'Banana', 'Orange']|length }}

{# Obtendo a quantidade de caracteres em uma string #}
{{ 'Hello, World!'|length }}
```
