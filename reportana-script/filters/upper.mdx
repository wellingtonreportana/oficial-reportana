### Introdução

O filtro `upper` é utilizado para converter todos os caracteres de uma string para maiúsculas. Esse filtro é útil quando se deseja padronizar o texto em um formato de caixa alta.

### Exemplo de uso

Exemplo de como utilizar o filtro `upper` para converter uma string em letras maiúsculas:

```twig
{{ 'Hello, World!'|upper }}
```
