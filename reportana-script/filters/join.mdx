### Introdução

O filtro `join` é usado para unir os elementos de uma lista ou array em uma única string, com um delimitador especificado entre cada elemento. Isso é útil para criar listas formatadas ou concatenar valores de uma coleção em um formato específico.

### Exemplo de uso

Exemplo de como usar o filtro `join` para concatenar elementos de uma lista com uma vírgula e um espaço como delimitador:

```twig
{{ ['apple', 'banana', 'cherry']|join(', ') }}
```
