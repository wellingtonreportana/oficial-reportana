### Introdução

O filtro `title` é utilizado para converter a primeira letra de cada palavra em uma string para maiúscula, deixando as demais letras em minúscula. Esse filtro é útil para formatar títulos ou nomes de forma padronizada.

### Exemplo de uso

Exemplo de como utilizar o filtro `title` para capitalizar cada palavra em uma string:

```twig
{% set text = 'hello world' %}
{{ text|title }}
```
