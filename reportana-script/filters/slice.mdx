### Introdução

O filtro `slice` é usado para extrair uma parte de uma string, array ou outro tipo de sequência. Ele permite que você selecione um subconjunto de elementos ou caracteres, especificando o início e a quantidade de elementos que deseja extrair.

### Exemplo de uso

Exemplo de como utilizar o filtro `slice` para obter os primeiros três caracteres de uma string:

```twig
{% set text = 'Reportana' %}
{% set sliced_text = text|slice(0, 3) %}
{{ sliced_text }}
```
