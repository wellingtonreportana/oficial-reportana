Podemos utilizar também os seguintes operadores para criar abordagens utilizando calculos matemáticos.

**Operadores matemáticos suportados atualmente:**

`+`: Adiciona dois números (os operandos são convertidos para números se necessário).

_Exemplo de uso:_

```twig
{# Adição #}
{{ 5 + 3 }} {# Retorna 8 #}
```

`-`: Subtrai o segundo número do primeiro.

_Exemplo de uso:_

```twig
{# Subtração #}
{{ 10 - 4 }} {# Retorna 6 #}
```

`/`: Divide dois números.

_Exemplo de uso:_

```twig
{# Divisão #}
{{ 10 / 2 }} {# Retorna 5 #}
```

`%`: Calcula o resto de uma divisão inteira.

_Exemplo de uso:_

```twig
{# Restante da divisão inteira #}
{{ 11 % 7 }} {# retorna 4 #}
```

`//`: Divide dois números e retorna o resultado inteiro arredondado para baixo.

_Exemplo de uso:_

```twig
{# Resultado inteiro arredondado #}
{{ 20 // 7 }} {# retorna 2 #}
{{ -20 // 7 }} {# retorna -3 #}
```

`*`: Multiplica o operando esquerdo pelo direito.

_Exemplo de uso:_

```twig
{# Multiplicação #}
{{ 2 * 6 }} {# Retorna 12 #}
```

`**`: Eleva o operando esquerdo à potência do operando direito.

_Exemplo de uso:_

```twig
{# Potência #}
{{ 3 ** 2 }} {# Retorna 9 #}
```
